# SHA-256
A SHA-256 é uma função hash que produz 32 bytes a partir de uma entrada de qualquer tamnho.

## /\ Propriedades
1. Determinística: uma mesma entrada terá uma mesma saída.
2. Irreversível: é impossível recuperar a entrada a partir do mesmo hash.
3. Resistente à colisões: é raro encontrar duas entradas com o mesmo hash.

## /\ Funcionamento:
1. Pré-processamento: é realizado um padding (são adicionados bits extras) para que o tamanho da mensagem original seja um múltiplo perfeito de 512 bits.
  Inicialmente, adiciona-se um bit '1' ao final da mensagem original. Após isso, acrescenta-se uma sequência de bits '0'. A quantidade de zeros adicionados deve ser a adeuada para que o tamanho da mensagem se torne múltiplo de 512 bits.
2. Divisão em blocos de 512 bits: o algotimo corta a sequência em blocos de 64 bytes.
3. Processamento: cada um dos blocos resultantes da fase anterior passa pelo seguinte processamento.
  a. Expansão da chave: o bloco de 512 bits é esticado para 64 palavras de 32 bits. São criadas 48 palavras a partir das 16 originais a partir de operações de rotação, shift e XOR. Ele é amassado e revirado para gerar mais dados para as próximas etapas.
  b. Laço principal: pega-se o estado atual e a palavra expandida da rodada e aplica uma função complexa que envolve operações lógics bit a bit, rotações ciruclares de bits, somas módulo 2^32 e constantes fixas (32 bits das raízes cúbicas dos primeiros 64 números positivos). Esse loop se repete 64 vezes para o mesmo bloco.
4. Atualização do estado: após a etapa anterior, o novo estado é somado ao estado anterior. Quando o último bloco é processado, o estado interno é concatenado para formar o hash final de 256 bits, que veremos como uma string hexadecimal de 64 caracteres.
